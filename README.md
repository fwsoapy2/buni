# Buni Boosting multi-page package

This package is set up as a small Express website with these routes:

- /
- /calculator
- /faq
- /discord

## Files

- `server.js` - Express server
- `public/assets/styles.css` - shared styles
- `public/assets/calculator.js` - calculator logic
- `public/index.html` - homepage
- `public/calculator/index.html` - calculator page
- `public/faq/index.html` - FAQ page
- `public/discord/index.html` - Discord handoff page

## Run locally

1. Install dependencies:
   `npm install`
2. Start the site:
   `npm start`

The server will run on port `3000` by default, or the value in `PORT`.
