# Buni Boosting - GitHub Pages build

This version is converted to a static multi-page website for GitHub Pages.

## Pages included

- Home
- Calculator
- Pricing
- FAQ
- Discord
- 404 page

## Upload steps

1. Open your `buni` GitHub repo.
2. Delete old backend files like `server.js` and `package.json`.
3. Upload everything inside this folder to the root of the repo.
4. Make sure GitHub Pages is set to deploy from `main` and `/(root)`.
5. Wait a minute, then open the site.

## Expected URLs for your current repo

- Home: `https://fwsoapy2.github.io/buni/`
- Calculator: `https://fwsoapy2.github.io/buni/calculator/`
- Pricing: `https://fwsoapy2.github.io/buni/pricing/`
- FAQ: `https://fwsoapy2.github.io/buni/faq/`
- Discord: `https://fwsoapy2.github.io/buni/discord/`

If you later connect a custom domain, the same page structure can become paths like `/pricing/` and `/faq/` on your own domain.
